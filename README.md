# Super Eats

A food ordering application similar to Uber Eats, built with Express.js backend and JavaScript frontend.

## Project Structure
```
Super_Eats/
├── backend/          # Express API with TypeScript
├── frontend/         # HTML/CSS/JavaScript
└── README.md
```

## Setup Instructions

### 1. Clone the Repository
```bash
git clone <repo-url from github>
cd Super_Eats
```

### 2. Backend Setup
```bash
cd backend
npm install
npm start
```

The API will run on `http://localhost:3000`.

### 3. Frontend Setup
```bash
cd ../frontend
# open index.html in your browser
# OR use Live Server extension in VS Code
```
